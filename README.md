# Burnout Budget (Expo Router + TypeScript)

A calm, non-judgmental budgeting companion focused on money stress + burnout patterns.

## Quick start
```bash
npm install
npx expo start -c
```

## Notes
- This project uses **Expo Router** (no root App.tsx / index.ts).
- Bank/Plaid + RevenueCat are stubbed for now (safe placeholders).
- AI chat uses a local, gentle fallback response unless you add a backend later.
